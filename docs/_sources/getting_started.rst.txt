Getting started
---------------

安装方法很简单：：

.. code:: python
   
   pip install ansys-mapdl-reader

你也可以访问 `pymapdl-reader <https://github.com/pyansys/pymapdl-reader>`_ 从 GitHub 下载源代码或最新的发布版本。

如果有任何安装（或其他）问题，请在 `pymapdl-reader Issues <https://github.com/pyansys/pymapdl-reader/issues>`_ 打开一个问题。

``ansys-mapdl-reader`` 支持 64 位 Windows、Mac OS 和 Linux，适用于 Python 3.7 至 Python 3.10。
