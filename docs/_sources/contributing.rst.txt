==========
Contribute
==========

为 PyAnsys 库做贡献的总体指导出现在 *PyAnsys Developer's Guide* 中的 `Contributing <https://dev.docs.pyansys.com/how-to/contributing.html>`_ 主题中。
在尝试贡献 PyMAPDL-Reader 之前，请确保您已经彻底熟悉了它和所有的 `Guidelines and Best Practices <https://dev.docs.pyansys.com/coding-style/index.html>`_ 。

以下是 PyMAPDL-Reader 特有的贡献信息。

Cloning the PyMAPDL Reader Repository
-------------------------------------
在开发模式下，运行此代码克隆并安装最新版本的 PyMAPDL-Reader：

.. code::

    git clone https://github.com/pyansys/pymapdl-reader
    cd pymapdl-reader
    pip install -e .


Posting Issues
--------------
使用 `PyMAPDL-Reader Issues <https://github.com/pyansys/pymapdl-reader/issues>`_ 页面提交问题、报告错误和申请新功能。

如需联系项目支持团队，请发送电子邮件至 `pyansys.core@ansys.com <pyansys.core@ansys.com>`_ 。

Viewing PyMAPDL-Reader Documentation
------------------------------------

PyMAPDL-Reader 最新稳定版本的文档托管在 `PyMAPDL-Reader Documentation <https://readerdocs.pyansys.com/>`_ 。
